# Jigsaw-Violated-Comments-Detection-Kaggle-Top-1-Solution
This solution is for Kaggle competition ''Jigsaw - Agile Community Rules Classification''. We utilized Qwen2.5-0.6B and Qwen2.5-14B model to do the classification, and used Qwen3.0-Embedding for semantic research. This solution got 0.914 on PB.
